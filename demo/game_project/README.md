# 宇宙之影 (Shadow of the Universe) - Demo

基于"物理妥协能力系统"的战役策略游戏原型演示

## 项目概述

这是一个创新的战役策略游戏概念，核心特色是**物理妥协能力系统**：
- 超能力是物理法则对某一方向的极致妥协
- 每次使用能力都有明确的代价和失败风险
- 现实债务累积会带来严重后果
- 核心玩法是风险管理而非资源管理

### 核心机制展示

1. **物理妥协技能系统**
   - 热量转移（违反热力学第二定律）
   - 静电聚集（违反库伦定律）
   - 反应增强（提升神经传导速度）

2. **风险管理机制**
   - 基础成功率：85-95%（低级血统）
   - 连续使用惩罚：每次-5%成功率
   - 债务过高惩罚：超过500点-10%
   - 疲劳惩罚：超过70点-15%

3. **现实债务系统**
   - 使用技能累积债务
   - 债务过高会影响技能成功率
   - 休息可以降低债务和疲劳
   - 债务达到1000触发危机

4. **AI决策系统**
   - 自动评估风险等级
   - 优先选择安全技能
   - 疲劳或债务过高时选择休息

## 核心设计亮点

### 1. 物理妥协概念

与传统RPG的"魔法值消耗"不同，本系统使用"物理法则妥协"概念：

```python
# 传统RPG技能
if mana >= 20:
    cast_fireball()
    mana -= 20

# 物理妥协技能
success_rate = calculate_modified_success_rate(user, skill)
if roll_random() <= success_rate:
    apply_effect()
else:
    suffer_backlash()
reality_debt += skill.cost
```

### 2. 动态成功率

成功率不是固定的，会受到多种因素影响：

```python
def get_modified_success_rate(self, user):
    rate = self.base_success_rate
    rate -= self.consecutive_uses * 5      # 连续使用惩罚
    rate -= 10 if user.debt > 500 else 0   # 债务惩罚
    rate -= 15 if user.fatigue > 70 else 0 # 疲劳惩罚
    return max(10, min(95, rate))          # 限制在10%-95%
```

### 3. 现实债务系统

创新的长期资源管理机制：

- **安全范围**（0-300）：无负面效果
- **注意范围**（300-500）：开始注意
- **警告范围**（500-700）：成功率降低
- **危险范围**（700-900）：高级技能禁用
- **极危范围**（900-1000）：可能触发危机

### 4. 风险等级分类

根据成功率自动分类风险等级：

- **安全**（85%+）：绿色，可以放心使用
- **低风险**（70-85%）：黄色，基本安全
- **中等风险**（50-70%）：橙色，需要谨慎
- **高风险**（30-50%）：红色，危险
- **极高风险**（<30%）：紫色，极危险

## 游戏设计理念

### 核心循环

```
风险评估 → 技能使用 → 结果判定 → 代价支付
    ↑                                      ↓
    └────────── 债务累积 ← 休息消散 ←──────┘
```

### 战术深度

1. **时机选择**：何时使用高风险能力？
2. **资源管理**：如何平衡债务和收益？
3. **风险控制**：何时该休息恢复？
4. **应急处理**：债务爆表时怎么办？

### 策略多样性

- **保守型**：多用基础攻击，少用技能
- **平衡型**：合理使用技能，注意休息
- **激进型**：频繁使用技能，追求高伤害
- **绝望型**：债务危险时仍使用技能

## 技术实现

### 数据结构

```python
@dataclass
class SkillEffect:

    failure_description: str      # 失败描述

    failure_damage: int           # 失败自伤
    reality_debt: int            # 现实债务代价
```

### AI决策逻辑

```python
def ai_decide_action(hero, enemy):
    if hero.fatigue > 70 or hero.debt > 600:
        return "休息"  # 保守策略

    safe_skills = [s for s in hero.skills
                   if s.get_risk_level(hero) == RiskLevel.SAFE]
    if safe_skills:
        return f"使用 {safe_skills[0].name}"

    return "基础攻击"  # 最后选择
```
