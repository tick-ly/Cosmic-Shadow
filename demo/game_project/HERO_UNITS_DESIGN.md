# 宇宙之影 - 英雄单位设计文档

## 📖 基于原文设定

本设计基于《无穷重阻》原文的核心设定：
- **阶段系统**：基础阶段 → 魔阶段 → 神阶段
- **血统能力**：基础/进阶/精英超能
- **能力类型**：预测、空间操控、时间操控、现实扭曲
- **代价系统**：现实债务、疲劳、反噬

## 🎮 能力系统设计

### 能力分类

#### 预测类 (Prediction)
- **战术预判**：预判敌方行动
- **战场感知**：全面态势洞察
- **命运凝视**：改变命运线

#### 空间类 (Spatial)
- **空间折叠**：瞬移
- **维度斩击**：无视防御攻击
- **空间锁链**：控制敌人

#### 时间类 (Temporal)
- **时间减缓**：减速效果
- **时间回溯**：撤销效果
- **时间冻结**：停止时间

#### 现实类 (Reality)
- **重力反转**：环境控制
- **因果改写**：改变因果
- **现实重构**：终极改造

#### 指挥类 (Command)
- **战术指挥**：团队协调
- **全军光环**：团队增益
- **战争统御**：全面控制

## 🎯 战术定位

### 各英雄适用场景

| 英雄 | 适合场景 | 不适合场景 |
|------|----------|------------|
| 预言者 | 策略战斗、信息战 | 正面硬刚 |

| 现实扭曲者 | 阵地战、破坏 | 快速机动战 |
| 指挥官 | 团队作战 | 单挑 |
| 均衡者 | 适应性强 | 需要专精 |
| 超越者 | 任何场景 | 无（全能） |

## 🔮 技能组合建议

### 预言者 + 指挥官
- 预测+指挥=完美战术配合
- 预言者提供信息
- 指挥官分配资源

### 空间使者 + 时间行者
- 空间+时间=时空操控
- 瞬移+减速=绝对控制

### 均衡者 + 任何英雄
- 均衡者的双重领域可配合任何英雄
- 提供额外的战术选择
