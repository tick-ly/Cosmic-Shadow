# 战斗系统升级总结

## 核心概念修正

根据原文《无穷重阻》设定，战斗系统已经从"单兵超能力战斗"升级为"战争机器+超能力辅助"。

## 正确的战斗概念

### 武器系统：战争机器

**大规模军事装备**：
- 航空母舰（主要火力平台）
- 要塞堡垒（防御阵地）
- 战列舰（海上主力）
- 机甲战士（机动单位）
- 重炮阵地（火力支援）
- 指挥中心（战术协调）

**战争机器特性**：
- 高生命值（数百到数千）
- 稳定的伤害输出
- 多种特殊系统（雷达、防空、舰载机等）
- 需要操作员指挥
- 能量容量限制

### 超能力：辅助增强

**主要用途**：
- **火力增强**：提升武器伤害输出
- **精度增强**：提高命中精度（雷达增强）
- **防御增强**：提升装甲和护盾
- **修复能力**：快速修复受损机体
- **射程增强**：扩大攻击范围
- **探测增强**：提升雷达和感知系统

**使用原则**：
- 能力有持续时间（1-4回合）
- 临时改变战局平衡
- 失败会导致机器受损
- 操作员承担现实债务

## 战术体验对比

### 之前的错误概念
- 超能力作为主要攻击手段
- 单兵战斗规模
- 直接伤害输出
- 类似传统RPG

### 正确的概念
- **武器**：稳定的伤害输出（88点/回合）
- **能力**：性能增强（精度+20%，持续4回合）
- **规模**：战争机器级别（航母vs舰队）
- **战术**：时机把握和风险管理

## 系统设计优势

### 1. 符合原文设定
- 参考原文中的航母、堡垒等大规模装备
- 超能力作为辅助增强系统性能
- 战争机器是主要战斗力

### 2. 战术深度
- **时机把握**：什么时候使用能力增强？
- **风险管理**：能力失败的后果可以承受吗？
- **资源管理**：操作员的现实债务累积
- **战术多样性**：不同能力的组合使用

### 3. 平衡性
- 武器提供稳定可靠的输出
- 能力提供临时性能提升
- 失败有明确代价（机体受损）
- 长期代价（现实债务）限制过度使用

### 4. 游戏循环
```
评估战况 → 能力增强（可选）→ 武器攻击 → 敌方反击
    ↑                                              ↓
    └── 战术休整（恢复状态）←── 机体受损/能量消耗 ──┘
```

## 扩展方向

### 多种战争机器

**航空母舰**：
- 特殊系统：舰载机群、防空系统、雷达系统
- 高生命值，远射程，中等伤害
- 需要大量船员操作

**要塞堡垒**：
- 特殊系统：防御系统、重炮群、指挥中心
- 极高生命值，固定位置，高伤害
- 防御力强，机动性差

**机甲战士**：
- 特殊系统：高机动性、多功能武器
- 中等生命值，高机动，中等伤害
- 单人操作，灵活性强

### 多种超能力增强

**低级血统能力**：
- 火力增强（+30伤害，3回合）
- 雷达增强（+20精度，4回合）
- 紧急修复（+50修复，即时）
- 探测增强（+50探测范围）

**中级血统能力**：
- 空间折叠攻击（瞬移打击）
- 时间减缓（提高反应速度）
- 能量护盾（大幅防御提升）
- 系统解锁（启用隐藏系统）

**高级血统能力**：
- 现实重构（改变战场环境）
- 因果逆转（改写战斗结果）
- 物理常数修改（根本性提升）

## 现代战争系统升级（2026-03-24）

### 最新改进

**1. 超能分类优化**
- ❌ 旧版本：低级血统/中级血统/高级血统（等级不平等）
- ✅ 新版本：基础超能/进阶超能/精英超能（平等称呼）

**2. 三级战争机器系统**

| 规模 | 生命值 | 装甲 | 探测范围 | 通讯范围 | 武器配置 |
|------|--------|------|----------|----------|----------|
| 单兵 | 100 | 20 | 200m | 100m | 步枪、机枪、狙击步枪 |
| 小队 | 500 | 40 | 800m | 500m | 机枪、自动步枪、反坦克导弹 |
| 军团 | 2000 | 60 | 3000m | 2000m | 155mm榴弹炮、防空系统、导弹发射井 |

**3. 指挥官光环系统**
- 精英指挥官提供军团范围buff
- 光环范围：5000m（覆盖整个战场）
- 光环效果：
  - 全军伤害+20
  - 通讯效率+40%
  - 探测范围+300m
  - 持续时间：5回合

**4. 现代武器配置**

单兵武器：
- 突击步枪：25伤，3发/回合，80%命中，400m射程
- 轻机枪：35伤，5发/回合，70%命中，600m射程，火力压制
- 狙击步枪：80伤，1发/回合，90%命中，800m射程

小队武器：
- 班用机枪：45伤，6发/回合，75%命中，800m射程，火力压制

- 反坦克导弹：150伤，1发/回合，60%命中，1000m射程，反装甲

军团武器：

- 战术导弹：300伤，1发/回合，70%命中，5000m射程，反装甲

**5. 信息战机制**
- 通讯效率：影响指令传达速度（单兵50% → 军团100%）
- 探测范围：影响敌方单位发现（200m → 3000m）
- 超能增强：可提升探测范围和通讯速度
- 战场感知：高等级单位提供更全面的信息
