## 能力分类系统

根据原文设定，能力分为以下类型：

| 类型 | 说明 | 示例能力 |
|------|------|----------|
| 感知型 | 侦查、预判类能力 | 广域扫描、战术预判 |
| 操控型 | 控制、操纵类能力 | 战术压制、双模式切换 |
| 强化型 | 增益、强化类能力 | 动力增幅、防御增幅、指挥光环 |
| 物质型 | 物质操控类能力 | 自适应护盾 |
| 能量型 | 能量相关类能力 | 共振打击、能量传输 |
| 信息型 | 信息处理类能力 | 信息同步、战场指挥 |
| 空间型 | 空间操控类能力 | 空间折叠、维度斩击 |
| 时间型 | 时间操控类能力 | 时间减缓、时间冻结 |

## 使用说明

```python
from hero_units_v2 import *

# 创建英雄系统
system = HeroSystem()

# 创建英雄
gamma = system.create_hero("gamma_7")

# 使用能力
result = system.use_ability(gamma, 0)  # 使用第一个能力

# 按级别获取
grade_a_heroes = get_hero_by_level(UnitLevel.GRADE_A)

# 按角色获取
striker_heroes = get_hero_by_role(HeroRole.STRIKER)
```
