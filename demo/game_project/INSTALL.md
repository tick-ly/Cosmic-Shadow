# 快速开始指南

## 立即体验Demo

### 方法1：自动演示（推荐）

最简单的方式，无需任何交互：

```bash
cd D:\program\univers\demo\game_project
python auto_demo.py
```

这将自动运行一场完整的战斗演示，展示所有核心机制。

### 方法2：交互式版本

如果你想亲自体验战斗决策：

```bash
python text_demo.py
```

然后按照提示选择行动：
- 输入 `1-3` 使用技能
- 输入 `0` 基础攻击
- 输入 `-1` 休息恢复

## 核心体验要点

### 1. 理解风险概念

每次使用技能都有成功率：
- **90%**：安全（绿色）
- **70-85%**：低风险（黄色）
- **50-70%**：中等风险（橙色）
- **30-50%**：高风险（红色）

### 2. 观察债务累积

注意查看[现实债务]数值：
- **0-300**：安全，随意使用
- **300-500**：注意，开始累积
- **500-700**：警告，成功率下降
- **700-900**：危险，部分技能禁用
- **900-1000**：极危，可能触发危机

### 3. 管理疲劳度

疲劳会影响成功率：
- **0-70**：正常状态
- **70-90**：疲劳，成功率-15%
- **90-100**：极度疲劳，无法使用技能

### 4. 连续使用惩罚

重复使用同一技能会降低成功率：
- 第1次：正常成功率
- 第2次：-5%
- 第3次：-10%
- 以此类推

## 战术建议

### 保守策略

1. 优先使用高成功率技能（85%+）
2. 疲劳超过70时休息
3. 债务超过500时转为基础攻击
4. 避免连续使用同一技能

### 平衡策略

1. 混合使用不同技能
2. 在安全范围内适度冒险
3. 定期休息恢复状态
4. 关键时刻承担风险

### 激进策略

1. 频繁使用技能追求高伤害
2. 接受较高的失败风险
3. 依赖运气和技能组合
4. 容易导致债务危机

## 常见问题

### Q: 为什么技能会失败？

A: 这是核心机制！每次使用技能都有失败概率，即使成功率90%也有10%失败可能。失败会受到反噬伤害，债务也会增加。

### Q: 如何降低失败风险？

A:
- 避免连续使用同一技能
- 控制债务和疲劳在安全范围
- 选择成功率高的技能
- 定期休息恢复状态

### Q: 债务太高怎么办？

A: 选择"休息"行动，每次休息可以：
- 恢复30点疲劳
- 减少20点债务

### Q: 游戏目标是什么？

A: 在债务不爆表（<1000）的情况下击败敌人，同时保持英雄存活。

### Q: 什么是完美胜利？

A: 在债务低于300的情况下击败敌人，表示优秀的风险管理。
