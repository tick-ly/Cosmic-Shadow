## 1. 技能系统框架

### 1.1 技能属性

```python
class Skill:
    """技能基础类"""

    def __init__(self):
        self.name = ""              # 技能名称
        self.category = ""          # 技能类别
        self.tier = ""              # 血统等级要求
        self.min_power = 0          # 最低战力值要求

        # 战斗属性
        self.damage = 0             # 造成伤害
        self.range = 0              # 作用范围
        self.duration = 0           # 持续时间
        self.aoe_radius = 0         # 范围半径

        # 消耗与限制
        self.energy_cost = 0        # 能量消耗
        self.cooldown = 0           # 冷却时间（秒）
        self.cast_time = 0          # 施放时间
        self.channeling = False     # 是否需要引导

        # 副作用
        self.side_effects = []      # 副作用列表
        self.instability_risk = 0   # 不稳定性风险

        # 战术价值
        self.strategic_value = 0    # 战略价值评分
        self.tactical_uses = []    # 战术用途

    def get_skill_card(self):
        """获取技能卡（游戏内显示）"""
        return {
            "name": self.name,
            "category": self.category,
            "tier": self.tier,
            "description": self.description,
            "effects": self.effects,
            "costs": self.costs,
            "requirements": self.requirements,
            "icon": self.icon
        }
```

class SkillUsageAI:
    """AI技能使用决策系统"""

    def __init__(self):
        self.usage_strategies = {}

    def decide_skill_usage(self, hero, battle_situation):
        """决定是否使用技能"""

        decision_factors = {
            "strategic_importance": battle_situation.importance,
            "hero_condition": hero.current_condition,
            "energy_level": hero.energy,
            "risk_assessment": self._assess_risks(hero, battle_situation),
            "expected_benefit": self._calculate_benefit(hero, battle_situation)
        }

        # 不同血统等级的决策逻辑
        if hero.bloodline_tier == "low":
            return self._low_level_decision(decision_factors)
        elif hero.bloodline_tier == "mid":
            return self._mid_level_decision(decision_factors)
        else:
            return self._high_level_decision(decision_factors)

    def _low_level_decision(self, factors):
        """低级血统：积极使用技能"""
        if factors["energy_level"] > 30:
            return {"action": "use_skill", "reasoning": "低风险，积极贡献"}
        else:
            return {"action": "conserve", "reasoning": "能量不足"}

    def _mid_level_decision(self, factors):
        """中级血统：战略性使用"""
        if factors["strategic_importance"] > 0.7 and factors["expected_benefit"] > factors["risk_assessment"]:
            return {"action": "use_skill", "reasoning": "关键时刻使用"}
        else:
            return {"action": "evaluate", "reasoning": "评估战术价值"}

    def _high_level_decision(self, factors):
        """高级血统：极端谨慎使用"""
        if factors["strategic_importance"] > 0.9:
            return {
                "action": "player_consultation",

## 总结

### 技能系统层次

```
低级血统技能 (70%觉醒者)
├─ 基础战斗增强
├─ 简单元素操控
├─ 感知提升
└─ 稳定可控

中级血统技能 (25%觉醒者)
├─ 空间操控能力
├─ 时间影响能力
├─ 精神力量
├─ 复合元素
└─ 需要训练维持

高级血统技能 (5%觉醒者)
├─ 现实重构
├─ 时间线操控
├─ 因果律影响
├─ 维度能力
└─ 极度危险，改变世界
```

### 战术价值

1. **低级技能**：战术优势，常规战斗
2. **中级技能**：战役转折点，改变战术
3. **高级技能**：战略武器，改变世界

### 使用哲学

- 低级血统：积极使用，贡献战场
- 中级血统：战略性使用，关键时机
- 高级血统：最后手段，玩家直接决策

这个系统为游戏提供了丰富的战术选择和深刻的道德考量。
