# 2km x 2km 战地风格地图草案

## 文件说明

本文件对应可视化草图：

- `battlefield-2km-layout.svg`

用途：

- 灰盒搭建参考
- 据点坐标与战线分区说明
- 载具路线与步兵绕后路线参考

## 总体尺寸

- 地图边界：`2000m x 2000m`
- 推荐模式：`16v16`
- 可扩展上限：`24v24`
- 据点数量：`5`
- 主战线数量：`3`
- 目标交战节奏：`30 秒内首次接敌`

坐标约定：

- 左下角为 `(0, 0)`
- 右上角为 `(2000, 2000)`
- 图中坐标和尺寸按米估算

## 据点布局

### A - Farm Outpost

- 坐标：`(360, 1520)`
- 类型：近距离攻防据点
- 周边环境：农场、矮墙、谷仓、沟渠
- 作用：西南出生后的第一落脚点

### B - West Town

- 坐标：`(720, 1150)`
- 类型：中近距离巷战据点
- 周边环境：小镇街区、断墙、两层建筑外壳
- 作用：中路西侧跳板

### C - Bridge Hub

- 坐标：`(1100, 1030)`
- 类型：主争夺核心据点
- 周边环境：桥头、河岸低地、沙包、简易工事
- 作用：全图节奏锚点

### D - Quarry Edge

- 坐标：`(1460, 820)`
- 类型：中远距离压制据点
- 周边环境：采石场、坡道、高差、裸岩掩体
- 作用：东北侧高地与载具线交汇点

### E - Orchard Yard

- 坐标：`(1500, 1280)`
- 类型：中距离据点
- 周边环境：果园、仓储区、围栏、装载点
- 作用：东南侧转场与反包抄节点

## 出生点

### Southwest Spawn

- 参考范围：`x=150~310, y=1580~1700`
- 保护设计：
  - 出生点前方先给掩体和缓坡
  - 不允许对面出生点直视
  - 至少保留两条离开路线

### East Quarry Spawn

- 参考范围：`x=1500~1680, y=620~780`
- 保护设计：
  - 通过采石场边缘和坡地挡视线
  - 出生后可在 `30 秒` 内接入 D 点外围

## 三条主战线

### 上路

- 路径：`东侧出生点 -> D -> 采石场高坡 -> 中路北岸`
- 特征：
  - 高差明显
  - 中远距离交战多
  - 适合压制与狙击

### 中路

- 路径：`B -> C -> D`
- 特征：
  - 全图最稳定交战区
  - 载具和步兵都会经过
  - 应优先做掩体密度和平衡

### 下路

- 路径：`A -> 南部农田 -> C 东南低地 -> E`
- 特征：
  - 更适合绕后和反推进
  - 掩体需要控制在“可推进但不龟缩”

## 载具路线

主载具路线建议 3 条：

1. `西南出生点 -> A -> B -> C`
2. `东侧出生点 -> D -> C`
3. `D -> C -> E`

设计原则：

- 载具可进入主战线，但不能直接压满所有步兵侧路
- 据点周围要设置硬掩体和载具转向成本
- 桥头与镇区道路要限制高速直冲

## 步兵绕后路线

建议 4 条隐蔽步兵线：

1. `A 北侧坡地 -> B 外围`
2. `B 西南低矮建筑群 -> C 河岸`
3. `D 北侧采石场边缘 -> C 高地侧`
4. `C 东南低地 -> E 果园内部`

设计原则：

- 绕后路线必须比主路更安全，但更慢
- 绕后路线不应允许无代价直冲出生点
- 每条绕后线至少有 2 次掩体跳点

## 区域尺寸建议

### 河道与桥头

- 河道有效宽度：`80m - 140m`
- 主桥区域宽度：`60m - 90m`
- 低地缓坡长度：`120m - 180m`

### 镇区

- 主要街区范围：约 `320m x 300m`
- 单条巷道宽度：`8m - 14m`
- 建筑组之间留出可视射线，但避免超长直线通道

### 采石场

- 占地建议：约 `500m x 420m`
- 高差建议：`20m - 35m`
- 至少提供两条上下坡路线

### 农田与果园

- 大块开阔地中必须插入田埂、围栏、废车或小棚屋
- 单段纯裸露移动距离不建议超过 `80m`

## 灰盒搭建顺序

1. 地形高差
2. 河道和道路
3. 据点圈与出生点
4. 大型硬掩体
5. 建筑壳体和围栏
6. 载具阻挡物
7. 视线检查和跑图检查

## 第一版灰盒模块清单

建议最少准备以下模块：

- `4m` 和 `8m` 矮墙
- 沙袋组
- 单层建筑壳体
- 双层建筑壳体
- 谷仓或仓库壳体
- 断墙模块
- 反载具路障
- 小桥头掩体
- 果园树列
- 废车与集装箱

## 性能预留

如果后面要进引擎制作，建议一开始就按以下规则放置：

- 地图切块按 `256m` 规划
- 远景山体与大建筑预留 LOD
- 近景掩体单独做碰撞
- 大面积植被默认不启用精细碰撞

## 当前草案适用阶段

这份地图适合：

- 灰盒阶段
- 单机战线验证
- 小规模联机验证

## 节奏校验

按步兵平跑 `6m/s` 估算，关键距离已压回目标范围：

- `Southwest Spawn -> A`：约 `170m`，约 `28s`
- `East Quarry Spawn -> D`：约 `178m`，约 `30s`
- `A -> B`：约 `516m`，约 `86s`
- `B -> C`：约 `399m`，约 `66s`
- `C -> D`：约 `417m`，约 `70s`
- `C -> E`：约 `472m`，约 `79s`
- `D -> E`：约 `462m`，约 `77s`

这使它更接近当前目标：

- 首次接敌控制在 `30 秒` 左右
- 相邻关键区转场控制在 `60-90 秒`

这份地图还不适合直接进入：

- 正式美术制作
- 大规模 `32v32` 以上联机
- 建筑破坏系统接入
