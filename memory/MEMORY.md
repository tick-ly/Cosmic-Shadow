## 使用说明

每个职业文件包含：
- **职业定义**：该角色在游戏开发中的定位
- **核心技能列表**：该职业需要掌握的技能
- **技能详细说明**：每个技能的具体应用场景与产出物
- **技能矩阵**：技能与常用工具的对应关系
