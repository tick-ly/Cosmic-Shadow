# 界面/体验设计师 (UI/UX Designer)

## 职业定义
界面/体验设计师负责游戏所有可视界面的设计与交互体验，让玩家能够高效地与游戏交互。

---

## 核心技能

### 1. HUD 设计 (Heads-Up Display)
设计游戏内实时信息显示。

**应用场景：**
- 设计核心信息显示：生命、能量、目标、小地图
- 设计 HUD 信息层级（重要/次要/可隐藏）
- 设计 HUD 美观性与功能性平衡
- 设计职业/场景差异化的 HUD 适配

**产出物：**
- HUD 设计稿
- HUD 信息层级定义文档
- HUD 适配说明文档

---

### 2. 菜单系统设计 (Menu System)
设计游戏菜单与导航。

**应用场景：**
- 设计导航结构（Tab 式/树状/扁平）
- 管理选项复杂度（不一次暴露所有选项）
- 设计控制器（手柄）适配
- 设计设置选项分类与默认值

**产出物：**
- 菜单导航结构图
- 菜单系统设计稿
- 手柄适配说明文档

---

### 3. 教程设计 (Tutorial Design)
设计新手引导体验。

**应用场景：**
- 设计渐进式教程（Progressive Tutorial）
- 设计上下文提示（Contextual Tips）
- 设计跳过与重置机制
- 设计教学关卡 vs 游戏内嵌入教程

**产出物：**
- 教程流程图
- 教程设计文档
- 教程完成率分析报告

---

### 4. 交互反馈设计 (Feedback Design)
设计用户交互的反馈机制。

**应用场景：**
- 设计按钮状态（Default/Hover/Pressed/Disabled）
- 设计操作反馈（选中、高亮、确认动画）
- 设计错误处理（输入错误提示、无效操作说明）
- 设计加载状态（进度条、占位符）

**产出物：**
- 交互反馈设计规范
- 按钮状态设计稿
- 错误处理说明文档

---

### 5. 提示与指引系统 (Guidance System)
设计游戏内提示与引导。

**应用场景：**
- 设计任务指引（自动寻路、箭头指引、地图标记）
- 设计功能解锁提示（新功能点亮）
- 设计异常状态提示（Debuff 图标、警告标识）
- 管理提示优先级

**产出物：**
- 提示系统设计文档
- 指引视觉规范
- 提示优先级配置表

---

### 6. 界面动效设计 (UI Animation)
设计界面的动画效果。

**应用场景：**
- 设计界面过渡动画
- 设计伤害数字/暴击特效
- 设计弹窗入出动画
- 控制动效性能（Overdraw 控制）

**产出物：**
- UI 动效设计稿
- 动效时长规范
- 动效性能评估报告

---

### 7. 本地化适配 (Localization)
确保界面适合多语言。

**应用场景：**
- 处理文字长度差异（德语 vs 中文）
- 设计 RTL（右到左）布局适配
- 设计图标文化差异
- 格式化货币/日期/单位

**产出物：**
- 本地化适配文档
- 多语言布局设计稿
- 本地化测试报告

---

### 8. 无障碍设计 (Accessibility)
确保游戏对所有玩家可玩。

**应用场景：**
- 设计字体大小可调
- 设计高对比度模式
- 设计色盲模式（Deuteranopia/Protanopia）
- 设计 UI 可缩放/透明度可调

**产出物：**
- 无障碍功能清单
- 无障碍设计规范
- 无障碍测试报告

---

### 9. 界面原型制作 (UI Prototyping)
快速验证界面设计。

**应用场景：**
- 使用 Figma/Sketch/Adobe XD 制作原型
- 制作交互原型（Principle/ProtoPie/Axure）
- 与开发对接：标注规范、切图规范
- 设计 UI 适配流程（多分辨率）

**产出物：**
- UI 原型
- 交互原型
- 标注文档、切图包

---

### 10. 数据可视化 (Data Visualization)
设计游戏内数据展示。

**应用场景：**
- 设计角色属性面板
- 设计战力/强度数值呈现
- 设计活动进度可视化
- 设计数据仪表盘（运营后台）

**产出物：**
- 数据可视化设计稿
- 数据面板设计文档
- 数据仪表盘设计稿
